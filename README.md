# E-Commerce Technical Challenge

## Product Management

This microservice is responsible for the administration and 
management of products in the system.

**Technologies**

> gradle version 8.5
>
> Spring boot 3.3.5
>
> jdk 17
> 
> postgrest 13.16


## Prerequisites
You need to have it installed

> git
> 
> gradle 8.5
> 
> jdk 17
> 
> postgrest 13.16

## Commands to use

### Build Project
```shell
gradle clean build
```

### Run
```shell
gradle bootRun
```
